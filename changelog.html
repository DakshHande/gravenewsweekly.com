Changelog - GraveNews Weekly

Version 1.2.0 (February 25, 2025)

  • New Feature: Added a real-time lag checker to monitor connection stability between users and the website.
  • Security Enhancement: Implemented stronger moderation and security measures for the lag-checking system, including timeout handling and error validation.
  • UI Improvements: Updated the CSS styling across all pages to ensure a consistent and professional look.
  • Performance Optimization: Improved website loading speed by optimizing image placeholders and reducing unnecessary scripts.

Version 1.1.0 (February 20, 2025)

  • New Page: Introduced the "Join Us" page, allowing users to join the GraveNews Academy team on Lichess and participate in daily chess tournaments.
  • Tournament Schedule: Added a table to display upcoming tournaments with dates, times, and join links.
  • Accessibility Improvements: Enhanced keyboard navigation and screen reader compatibility for better accessibility.

Version 1.0.0 (February 15, 2025)

  • Initial Release: Launched the GraveNews Weekly website with the following features:
    • Home page with the latest chess news and articles.
    • Image placeholders for articles with customizable links.
    • Responsive design for seamless viewing on all devices.
